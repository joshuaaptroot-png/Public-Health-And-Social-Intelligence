PROJECT GOAL: 

This project is for the professional development of data analysts, Social Epidemiologists and data engineers - the aim is to experiment with publicly available data from the NHS, ONS, BNF and SNOMED datasets, aiming to extract real insights using raw data. 

Version control: 
please pull existing repository and create a new branch. before merging new branch with main, please pull to main to check for conflicts, if no conflicts then push to main. If conflicts please consult with other participants. 

PROJECT TECH STACK: 

The project employs a homemade Postgres SQL server and a DBeaver connection for data storage and ETL design. The DB is structured around medallion architecture. Modelled data will be pushed to repository repository via .CSV for data analysts to extract information, all .PBIX files will be uploaded for share insights and skill swapping between participants.
